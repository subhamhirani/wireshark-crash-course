## What's Inside

| Feature | Details |
|---|---|
| Live packet capture | 20 simulated packets — DNS, HTTP, TCP handshake, ARP, ICMP, DHCP, TLS, port scan |
| Wireshark filter engine | Type real filters (dns · tcp · ip.addr == x.x.x.x) and see live results |
| 12 essential filters | With use-case explanations and Hindi notes 🇮🇳 |
| 7 protocol decoders | DNS, HTTP, TCP, ICMP, ARP, TLS, DHCP — field-by-field breakdown |
| 8-question quiz | Scenario-based questions for N+ / CCNA / SOC analyst interviews |
| Fully offline | No backend, no API — single HTML file, runs in any browser |

## Skills Demonstrated

- Network protocol analysis (OSI Layers 2–7)
- Wireshark display filter syntax
- TCP 3-way handshake, port scan detection, ARP spoofing detection
- HTTP vs HTTPS security differences
- DNS, DHCP, ICMP, TLS protocol internals
- SOC analyst threat detection concepts

## How to Use

1. Click the [Live Demo link](#live-demo) above
2. **Tab 1 — Live capture**: click any packet row to see all layers. Type a filter in the box.
3. **Tab 2 — Filters**: click any card to apply that filter immediately
4. **Tab 3 — Protocol decoder**: select a protocol to see every field explained
5. **Tab 4 — Quiz**: test your knowledge with 8 scenario-based questions
